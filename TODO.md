## Features per Page

1. Landing Page - Hero, features, teacher showcase, CTA, footer
2. Student Dashboard - Sidebar, course cards, recent lessons, notifications
3. Teacher Dashboard - Stats cards, course management, create button
4. Courses Page - Search, filters, course grid
5. Course Details - Header, lessons, enroll button, reviews
6. Course Viewer - Dark theme, video placeholder, custom controls, sidebar
7. Community - Post creation with image preview, feed, interactions
8. Profile - Header, courses, posts
9. Settings - Editable form, image upload preview
